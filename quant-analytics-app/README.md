# Quant Analytics Platform  
**Custom Financial Analytics Dashboards & APIs**

---

## Overview

This repository contains a **production-ready Quant Analytics Platform** designed for financial market analysis using historical OHLCV data.

The platform is built as a **single integrated system**, not a collection of demo projects.  
It combines a scalable backend with interactive dashboards and is suitable for real-world trading, portfolio analysis, and fintech analytics use cases.

---

## Key Features

### Market Price & Returns Analytics
- Historical OHLCV analysis
- Daily, cumulative, and total returns
- Performance summaries via clean JSON APIs

---

### Multi-Asset Comparison Engine
- Cross-asset performance comparison
- Volatility analysis
- Correlation matrix visualization

---

### Technical Indicators Dashboard
- SMA / EMA
- Momentum & ROC
- ATR and trend-based indicators
- Parameterized analytics

---

### Risk & Volatility Analysis
- Volatility metrics
- Maximum drawdown
- Sharpe & Sortino ratios
- Value at Risk (VaR / CVaR)

---

### Portfolio Analytics
- Multi-asset portfolio analysis
- Weighted returns
- Portfolio volatility
- Correlation & diversification metrics

---

### Strategy Backtesting Engine
- Rule-based strategies (e.g. moving average crossover)
- Signal generation
- Equity curves
- Drawdown & performance statistics

---

### Market Regime & Risk State Detection
- Trend strength evaluation
- Volatility clustering
- Market regime classification
- Regime timelines

---

### Earnings Event Impact Analysis
- Pre-earnings vs post-earnings returns
- Gap analysis
- Volatility expansion around earnings

---

### Price-Based Sentiment Engine
- Momentum strength
- Volume anomalies
- Price–volume divergence
- Sentiment classification (Bullish / Neutral / Bearish)

## Architecture

### Backend
- FastAPI
- RESTful HTTP APIs
- Modular analytics engines
- Redis caching & rate limiting
- PostgreSQL-ready persistence layer

### Frontend
- Streamlit dashboards
- Symbol-based tab navigation
- Tables and interactive charts (Plotly)
- Clean separation of data and presentation

### Infrastructure
- Docker & Docker Compose
- Environment-based configuration
- VPS / cloud deployable
- Scalable and maintainable architecture
